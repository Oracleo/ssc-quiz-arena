# Privacy Policy

Last updated: March 17, 2026

SSC Quiz Arena is an educational quiz app for exam preparation.

Information we may process:
- Quiz progress, XP, streaks, and theme preference stored on your device.
- Basic account profile details only if optional sign-in is enabled in a future release and you choose to use it.

How data is used:
- To save progress and preferences.
- To improve app stability and user experience.
- To support optional sync features when available.

Local storage:
- The current production release stores progress and preferences on your device.

Cloud storage:
- Cloud sync is not enabled in the current Android production release unless it is explicitly configured in a later update.

Data sharing:
- We do not sell personal data.
- Platform services such as Google Play may process limited operational data required to deliver the app.
